# Yum Repositories

Installs yum repositories


## OS Support

* CentOS 6.x
* CentOS 7.x


## State Files

* `epel.sls`

    * Adds the EPEL repository

* `saltstack.sls`

    * Adds the official Saltstack repository

* `webtatic.sls`

    * Adds the Webtatic repository

* `percona.sls`

    * Adds the Percona repository
